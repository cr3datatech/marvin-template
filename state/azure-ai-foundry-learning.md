# Azure AI Foundry (Microsoft Foundry) Learning Plan

## What It Is
- **Official Name (2026)**: Microsoft Foundry
- **Tagline**: "The AI app and agent factory - build, optimize, and govern AI apps and agents at scale"
- **Purpose**: Platform for building, optimizing, and governing AI applications and agents
- **Audience**: Cloud architects, AI engineers, app developers

## Key Areas to Learn
1. **App Building** — Creating AI-powered applications
2. **Agent Building** — Creating autonomous or semi-autonomous AI agents
3. **Optimization** — Performance tuning, cost management
4. **Governance** — Security, compliance, monitoring, audit trails
